Relembrando Conceitos:

Valor é um espçao reservado na memória para armazenar uma informação

As variáveis podem ser de valores numéricos ou alfabéticos,não sendo preciso declarar de que tipo é a variável.

tipos de varáveis

int:inteiro

str:string,é uma sequência de caracteres simples,são utilizadas com aspas simples ‘exemplo’,ou aspas duplas “exemplo”.

list:- Listas agrupam um conjunto de elementos variados,podendo conter: inteiros, floats, strings, outras listas e outros tipos.

Tuple:- A diferença para Lista é que, após sua definição Tuplas não podem ser modificadas.

Dic: Dicionarios são utilizados para agrupar elementos através da estrutura de chave e valor, onde a chave é o primeiro elemento seguido por dois pontos e pelo valor.

Operadores de atribuição

=: operador de atribuição. Dada uma variável x, ao fazermos x = valor atribuímos o valor à variável x.

+=: é equivalente a fazer x = x + valor.

-=,=, /=, //=, %=,*=: funcionam da mesma forma que o +=
