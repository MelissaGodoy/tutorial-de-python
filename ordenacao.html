Ordenação

Com os algorítimos de ordenação podemos organizar os elementos de acordo com nossas necessidades (ordem alfabética,ordem crescente,ordem decrescente e etc)

Existem vários algorítimos de ordenação e é de suma importância que se faça a escolha certa em sua aplicação. Para isso, veremos a seguir um pouco sobre cada algorítimo:

Bubble Sort

É o algorítimo mais simples,ele compara um elemento com o seu próximo e de acordo com a sua necessidade verifica se é preciso realizar a troca ou não.

Ex:

Selection Sort

É uma melhoria do Bubble Sort,pois compara um elemento ja com todos os outros para no final realizar a troca,deixando a busca mais agil.

Insertion Sort

É um algorítimo silmples e eficiente se aplicado corretamente.

A lista é percorrida da esquerda para a direita deixando os elementos a esquerda ordenados.

Comb Sort

O Comb Sort também é parecido com o Bubble Sort,porém é mais eficiente pois consegue comparar elementos mais distantes entre si.

Para saber qual o intervalo basta dividir o tamanho da lista por 1,3 e arredondar p um número inteiro.

Heap Sort

O Heap Sort simula uma árvore binária completa com pais e filhos,sendo que os filhos são sempre menores que os pais

Shell Sort

o Shell Sort nada mais é do que uma melhoria do Insertion Sort,quebrando a lista em sublistas menores.

Merge Sort

O Merge Sort é uma melhoria do Insertion Sort,quebrando a lista em sublistas

Quick Sort

O Quick Sort é um algoritimo para divisão e conquista.

Ele tem o poder de separar uma sequencia de elementos com base em um.

Seu funcionamento baseia-se no Particionameto,particionar significa escolher um número como pivot e colocá-lo em uma posição tal que todos elementos à esquerda são menores ou iguais a ele,e todos os elementos à direita são maiores.
